# Responsive Profile Card

A modern and fully responsive profile card built using **HTML5** and **CSS3**.  
This project demonstrates the use of Flexbox, pseudo-elements, responsive design techniques, and Font Awesome icons.

## 🚀 Features

- Responsive layout (mobile-friendly)
- Clean and modern UI design
- Flexbox-based alignment
- Custom social media icons using Font Awesome
- Hover effects on buttons
- Analytics section with icons and counters
- Profile image with circular border effect

## 🛠️ Technologies Used

- HTML5
- CSS3
- Flexbox
- Font Awesome (CDN)

site is live at https://mohsinkhan4050.github.io/Responsive-Profile-Card/
